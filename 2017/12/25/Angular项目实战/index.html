开发准备

安装

1.在项目目录下：

安装第三方依赖库：npm install jquery –save

npm install bootstrap ——save

2.在.angular.cli.json中 添加

“Style”:[

“styles.css”,

​ “../node_modules/bootstrap/dist/css/bootstrap.css”

]

“scripts”: [

​ “../node_modules/jquery/dist/jquery.js”,

​ “../node_modules/bootstrap/dist/bootstrap.js”

​ ],

3.在typescript中不能编译 $ , 需要安装 npm install @types/jquery –save-dev

npm install @types/bootstrap –save-dev

安装的目的是让typescript识别jquery，bootstrap

组件可重用性非常重要

依赖注入：一种设计模式

什么是依赖注入模式及使用依赖注入的好处

依赖注入：Dependency Injection 简称 DI 控制反转：Inversion of Control 简称 IOC

A依赖B，B可以依赖注入，而不需要自己

一种松耦合的，可重用性，可测试性

在@ngModule({

Providers:[ProductService]

Providers里是需要依赖注入的

})

介绍Angular的依赖注入实现注入器和提供器
